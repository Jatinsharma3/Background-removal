# 🖼️ BG Removal App

A simple full-stack web app for removing image backgrounds using an external API. Built with **React** (frontend), **Node.js + Express** (backend), with **Clerk** for authentication and **Razorpay** for payments.

## 🚀 Features

- ✅ Remove image backgrounds via external API
- ✅ User authentication using Clerk
- ✅ Payments integrated with Razorpay
- ✅ Upload image → Remove background → Download transparent PNG
- ✅ Built with React, Node.js, and Express

## 🛠️ Tech Stack

- **Frontend**: React
- **Backend**: Node.js, Express
- **Auth**: Clerk
- **Payments**: Razorpay
- **Image Processing**: External API

## 📸 Use Case

Perfect for:
- E-commerce product images
- Profile photos
- Content creators
- Quick background cleanup
